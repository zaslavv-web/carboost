## Слайд 1 · Нижние 4 блока → плитка 2×2, увеличенные вдвое

**Файл:** `src/pages/investor/slides/Slide1Product.tsx`

### Что меняется

Сейчас 4 карточки («Для компаний со 100+ сотрудников», «Одно окно для сотрудника», «100% российская разработка», «Облако или установка в контуре») лежат в один ряд по 4 колонки — узкие, текст обрезается.

Переделываю в **сетку 2×2** (плитка), где каждая карточка занимает ~в 2 раза больше площади:

```text
┌──────────────────┬──────────────────┐
│  Для компаний    │  Одно окно       │
│  со 100+         │  для сотрудника  │
├──────────────────┼──────────────────┤
│  100% росс.      │  Облако / контур │
│  разработка      │                  │
└──────────────────┴──────────────────┘
```

### Конкретные правки

- Контейнер: `grid-cols-4 gap-4` → `grid-cols-2 gap-8`, каждая карточка `col-span-1` (вместо 1/4 ширины становится 1/2 → +100% по ширине).
- Внутренние отступы карточки удваиваются: `p-6` → `p-12`.
- Иконка: контейнер `w-12 h-12` → `w-30 h-30`, размер иконки `w-6 h-6` → `w-12 h-12`.
- Заголовок карточки: `text-[20px]` → `text-[32px]` (в пределах 11–60px).
- Текст описания: `text-[14px]` → `text-[22px]`, `leading` подкрутить под новый размер.
- Расстояния между иконкой/заголовком/текстом (`mb-3`, `mt-2` и т.п.) — удвоить.
- Всё остаётся редактируемым (`<Editable>`), тексты и ключи не трогаю.

### Проверки

- Убедиться, что новая плитка помещается по высоте 1080px вместе с верхним блоком (заголовок + подзаголовок + логотип). При необходимости чуть уменьшить верхний отступ секции.
- Светлая тема (#F7F4EC фон, белые карточки, золотая рамка) сохраняется.
- Минимальный шрифт ≥ 11px, максимальный ≤ 60px — соблюдено.

Никаких других слайдов и файлов не трогаю.
