## Презентация для HRD — «Career Track / Пик Роста: обновлённая версия»

### Формат и дизайн

- **PPTX**, 16:9, 1920×1080.
- Палитра проекта **Ocean Deep**: navy `#0c2340`, mid `#1a4a6e`, teal `#2d8a9e`, mint `#5cbdb9`, bg `#f5f9fb`.
- Типографика: заголовки Calibri Bold (как ближайший к Urbanist «системный»), body Calibri. Кикеры uppercase 22pt с трекингом.
- Композиция «sandwich»: тёмные тайтловые слайды (cover, разделители, финал), светлые контентные.
- На каждом слайде: kicker + крупный заголовок + 1 визуальный акцент (иконка-кружок, стат-карта, таблица сравнения)  
общее количество слайдов не должно быть больше 6

### Технический подход

- Генерация через `pptxgenjs` (CLI `npm install -g pptxgenjs` уже доступен в среде).
- Скрипт под `/tmp/build_hrd_deck.js`, файл результата — `/mnt/documents/Pik-Rosta-HRD-Overview.pptx`.
- Контент 16 модулей беру из `src/data/features.ts` + i18n RU.
- QA: после генерации конвертирую в PDF через LibreOffice, рендерю каждую страницу в JPG, читаю их и проверяю на overflow / overlap / контраст. Фиксирую и пересобираю до чистого прогона.
- Артефакт выдам через `<presentation-artifact>`.

### Открытые допущения (фиксирую по умолчанию, скажите если поменять)

- Сроки: облако **2-8 недель**, on-prem **8-24 недели** (как вы указали).
- Сравниваем сразу с двумя сегментами конкурентов (российские + зарубежные) — 2 отдельных слайда.
- В сравнении не критикуем конкурентов лично, только feature-grid.
- Без цен и SLA — позиционная презентация, не коммерческое предложение.
