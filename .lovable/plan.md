
## Что я понял из твоего фидбека

Я не сделал главного — не оставил **место под изображение** в макете каждого слайда. Иллюстрации я лепил "поверх" абсолютным позиционированием, а не в сетку. Также нарушил единство шрифта (Instrument Serif + системный) и не привёл в порядок таблицу на слайде 5. Ниже — точечный план, слайд за слайдом.

## 0. Глобально (общий шрифт и токены)

- Подключить **одну гарнитуру без засечек с табличными (моноширинными) цифрами** — вариант: **"Golos Text"** (кириллица, tabular-nums, цифры не «плывут»). Century Gothic я предложу как альтернативу, но у неё нет полноценной кириллицы в вебе — Golos ближе всего по духу и с цифрами всё в порядке. Если ты настаиваешь — подключу Century Gothic для латиницы + Golos для кириллицы.
- Убрать `Instrument Serif` из всех слайдов. Заголовки — тот же шрифт, крупнее и с `font-weight: 700`, курсив только там, где нужен смысловой акцент (`font-style: italic`).
- Включить `font-variant-numeric: tabular-nums` глобально в `.slide-content`, чтобы цифры в таблицах и KPI не прыгали.
- Заменить символ `₽` во всех местах на постфикс ` руб.` / ` тыс. руб.` / ` млн руб.`
- Уменьшить базовый размер тела на **~20%** там, где ты указал (слайды 4, 5, 7, 8): было 24–28px → станет 20–22px; заголовки 64px → 52px.

## 1. Слайд 1 — сетка «текст + место под изображение»

Пересобрать в 2-колоночную сетку по твоему скетчу:

```text

│  Логотип + название          │                  │
│  H1 заголовок                │   ИЗОБРАЖЕНИЕ    │
│  Подводка                    │   (peak/growth)  │
│                              │                  │
├──────────────┬───────────────┴──────────────────┤
│  Карточка 1  │  Карточка 2                      │
├──────────────┼──────────────────────────────────┤
│  Карточка 3  │  Карточка 4                      │
└──────────────┴──────────────────────────────────┘
```

Изображение — в правой колонке верхнего блока, а не абсолютом поверх текста.

## 5. Слайд 5 — таблица (главная переделка)

Сейчас в колонках намешано. Сделаю строго:

| Наименование (лево) | Стоимость (право) |
|---|---|
| 2 × Разработчик (fullstack) | 500 тыс. руб./мес. |
| 1 × Технический лидер | 500 тыс. руб./мес. |
| … | … |
| **Итого за год** | **19 800 тыс. руб.** |

- Заголовок колонки «Тыс. руб./мес.» — **над числовой колонкой**, а не в правом верхнем углу карточки.
- Единицы измерения — часть значения в правой колонке, а не отдельная плашка сверху.
- Все шрифты × 0.8 (тело 24→20, заголовок 26→22, крупная сумма 72→58).
- Символ `₽` убрать везде → `руб.` в конце.
- Пончиковую диаграмму сделать компактнее, легенду — под ней, а не сбоку.
- Добавить лёгкую полосатую заливку строк (`nth-child`) — визуально «читаемая таблица».

## 6. Слайд 6 — верстка по гайдам Android (Material 3)

- Material-карточки: `elevation-1`, скругление 12px, отступ между карточками 16px, внутренний padding 24px.
- Типографика по Material scale: `display-small` для заголовка, `title-medium` для заголовков карточек, `body-large` для текста.
- Один основной accent-color (золото бренда) + нейтральные surface-цвета.
- Использовать **Segmented Buttons / Chips** для этапов воронки продаж вместо текущей верстки.
- Иконки — outlined в стиле Material Symbols (через lucide `strokeWidth={1.5}`).

## 7. Слайд 7 — таймлайн умещается в плашку + анимации

- Таймлайн переверстать: горизонтальная линия с 4–5 точками, каждая точка = плашка фиксированной высоты, текст обрезается по границам.
- Шрифт × 0.8.
- К каждому этапу — **анимированная инфографика** (framer-motion): пульсирующая точка, растущая полоса прогресса, счётчик чисел, «печатающийся» текст статуса. По одной идиоме на этап, не мешанина.

## 8. Слайд 8 — аналогично слайду 7

- Шрифт × 0.8.
- Каждый пункт роадмапа/статуса получает свою анимацию (появление по scroll-in, счётчик, чек-марк с обводкой).
- Все пункты гарантированно в границах плашки (max-height + overflow-hidden с корректной вёрсткой).
