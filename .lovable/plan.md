## Презентация для HRD (.pptx, 6 слайдов)

Сгенерирую файл через `pptxgenjs` и сохраню в `/mnt/documents/career-track-hrd.pptx`. Палитра — фирменная teal/slate (под бренд Career Track), шрифт Calibri/Arial, плотность по skill/pptx (заголовки 40-54pt, body 20-24pt, крупные статы 72-120pt).

### Структура слайдов

1. **Титул** — «Career Track для HRD: меньше боли, больше доказуемого результата». Подзаголовок про 6 ключевых болей HRD и ROI для совета директоров.

2. **6 болей HRD** (2×3 сетка карточек) — основа презентации:
   - Удержание ключевых сотрудников
   - Объективная оценка компетенций
   - Понятный путь роста
   - Мотивация без роста ФОТ
   - Быстрый онбординг
   - Решения на основе данных
   
3. **Как Career Track закрывает боли** (таблица: Боль → Инструмент → Эффект) — связка болей с фичами (AI-оценка, треки, геймификация, паспорт, онбординг, аналитика).

4. **Цифры для руководителя** (3 крупных стат-блока) — конкретные метрики из кейсов:
   - −24% текучесть за 6 мес.
   - +37% вовлечённость, −18% бюджет премий
   - 30+ часов экономии в квартал на оценках

5. **Как обосновать руководителю** — структура аргументации: проблема в деньгах → решение → измеримый KPI → срок окупаемости. Готовые формулировки для защиты бюджета (стоимость текучести, ФОТ vs геймификация, часы HR vs автоматизация).

6. **Следующий шаг** — что HRD делает завтра: пилот на 1 подразделении, метрики до/после за 90 дней, презентация совету директоров через квартал. CTA + контакт.

- Запуск через `code--exec` с inline JS (pptxgenjs).
