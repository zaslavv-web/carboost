# Career Track

HR-tech платформа развития сотрудников.

- **Фронтенд** — React 18 + Vite + Tailwind (`src/`)
- **Бэкенд** — Laravel 11 + Sanctum (`backend-laravel/`)
- **БД** — MySQL/PostgreSQL (миграции `backend-laravel/database/migrations/`,
  данные — `backend-laravel/database/dumps/`)

## Запуск фронта

```bash
bun install
bun run dev
```
